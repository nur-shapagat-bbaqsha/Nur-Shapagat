# Нұр-шапағат бөбекжай балабақшасы — бір HTML ішіндегі стиль және скрипт

Бұл ZIP ішіндегі әр `.html` файлда CSS және JavaScript бірге тұр. Сондықтан `style.css` және `script.js` қажет емес.

## Қалай өзгерту керек
Әр HTML файлдың төменгі жағындағы `<script>` бөлімінен мыналарды табыңыз:

- `LINKS` — басты бет фотолары
- `DATA` — PDF құжат, фото және YouTube сілтемелері

Тек `[GOOGLE_DRIVE_LINK]`, `[YOUTUBE_LINK]`, `[PHOTO_...LINK]` орнын өз сілтемеңізге ауыстырыңыз. Қалған кодты өзгертпеңіз.

Басты беттегі балабақша фотосы сіз жіберген Google Drive сілтемесімен қойылды.

Google Drive файлдары ашылуы үшін: Share → General access → Anyone with the link → Viewer.


Жаңарту:
- Барлық беттерден Жүктеу батырмасы алынды. Құжаттар тек Қарау арқылы ашылады.
- Басты беттегі балабақша фотосы Google Drive preview арқылы қойылды.
