# Нұр-шапағат бөбекжай балабақшасы

Бұл ZIP ішінде сайттың жаңартылған нұсқасы бар.

## Не өзгерді
- `index.html` соңғы дұрыс нұсқа ретінде қалдырылды.
- `Жетістіктер` бөлімі ішкі беттерге бөлінді:
  - Бала жетістіктері
  - Педагогтер жетістіктері
  - әрқайсында 2023-2024, 2024-2025, 2025-2026 беттері бар.
- `Әдістемелік құжаттар` бөлімі ішкі беттерге бөлінді.
  Мысалы, `Циклограммалар` басқанда PDF ашылмайды, `methodical-cyclograms.html` деген бөлек бет ашылады.
  Сол бетте жылдар бойынша PDF карточкалар тұрады.
- Барлық стиль мен JavaScript әр HTML файлының ішінде.
- Құжаттарда тек `Қарау` батырмасы бар.

## Сілтемені қалай ауыстыру керек
Әр ішкі бетте төменгі жақта мынадай массив бар:

```js
const DOCUMENT_GROUPS = [
  {
    year: "2023-2024 оқу жылы",
    documents: [
      { title: "Құжат атауы", link: "[GOOGLE_DRIVE_LINK]" }
    ]
  }
];
```

Тек `title` және `link` өзгертіңіз.


## Жаңарту: әр құжатқа жеке ссылка
Барлық PDF құжаттар енді мына форматта берілді:

```js
{
  title: 'Құжат атауы',
  link: '#',
  type: 'pdf'
}
```

`#` тұрған жерге Google Drive PDF сілтемесін қойыңыз. Егер `#` қалса, сайтта батырма `Сілтеме жоқ` болып шығады және ашылмайды.

Мысалы:

```js
{
  title: 'Жарғы',
  link: 'https://drive.google.com/file/d/FILE_ID/view?usp=sharing',
  type: 'pdf'
}
```
